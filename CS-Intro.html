CSCI 111 Guide

Many students marjoring in Computer Science will often take CSCI 111 as their first course. This course also requires students to have access to a C++ compiler which is provied to them on the MARS server.

This guide will go over how to use the MARS server along side Visual Studio Code.

Part 1 - Installing Visual Studio Code (VSCode)

This part should be faily straight foward. Go to code.visualstudio.com and follow the installation instructions. Once you have installed and opened VSCode click on the button on very bottom left of VSCode. Then press "Connect To Host", then "Add new SSH Host"
