16 bit RISC CPU

I have built a 16 bit CPU RISC using logisim-evolution. It's instructions can be easily assembled by hand and features a TTY with keyboard, RGB display and Telnet.

Rust Calculator

This is my first project written in Rust. It is a simple graphical calculator that uses GTK as it's graphics library. This project taught me a lot about rust, with me learning new concepts such as borrowing. The biggest challenge I had was when I was trying to figure out how to bind GTK's graphical event signals to other parts of my code. I used concepts I learned in my Java course, such as Model, View, Controller.

HTTP webserver build in Rust

This is my first backend rust project. It uses libraries native to rust like TcpListener and TcpStream. It supports basic html & css pages and can load other file formats for photos and documents. It includes a 404 page not found handler as well.
